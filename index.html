Predictable scripting, zero setup

Build scripts that are secure by default and easy to ship.

SCALF is a scripting language focused on stability, permission-aware runtime behavior, and practical tooling for formatting, linting, testing, and standalone builds.

Run It Read Spec

Security first

Granular runtime permissions for file system, network, and environment access.

Developer tooling

Built-in `fmt`, `check`, `test`, docs generation, and optional VM execution.

Deployable output

Compile scripts into standalone binaries with feature-aware build minimization.

Quickstart

Run SCALF in minutes

cargo test
cargo run
cargo run -- docs
cargo run -- fmt --check path/to/file.rask
cargo run -- check path/to/file.rask
cargo run -- test path/to/file.rask
cargo run -- --vm path/to/file.rask
cargo run -- build path/to/file.rask --target=linux-x64
cargo run -- startup path/to/file.rask --iterations=25
